## Старт проекта

1. Установка зависимостей 
    ```bash
    $ yarn install
    ```
2. Запуск проект
    ```bash
    $ yarn start
    ```


## Стек продукта

- [@emotion](https://github.com/emotion-js/emotion/tree/main )
