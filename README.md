## Старт проекта

1. Установка зависимостей 
    ```bash
    $ yarn install
    ```
2. Запуск проект
    ```bash
    $ yarn start
    ```


## Стек продукта
- [react](https://reactjs.org/)
- [@emotion](https://github.com/emotion-js/emotion/tree/main )
- [mui-react](https://github.com/mui/material-ui)
